Install TallyInsight Agent

To connect your Tally data securely, you need to install our local agent. The agent runs on your computer and syncs data without exposing your Tally system.

Step 1: Download Agent

Download EXE Download BAT

Recommended: EXE for most users.

Step 2: Run Agent

  1. Close Tally if it’s open
  2. Right-click the downloaded file
  3. Select Run as Administrator
  4. Open Tally

Step 3: Sync Company

The agent will guide you to select your Tally company and start syncing.

Keep this page open — it will automatically detect when the agent is running.
